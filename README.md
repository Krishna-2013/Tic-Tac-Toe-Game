## 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project demonstrates core front-end development skills like DOM manipulation, game logic handling, and responsive UI design.

## 🚀 Features

* Two-player gameplay (Player X vs Player O)
* Real-time turn switching
* Win detection logic
* Draw detection
* Game reset functionality
* Clean and responsive UI
* Sound effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

📂 Project Structure
